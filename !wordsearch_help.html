!wordsearch Help

How to Use

Simply enter the word you want to look up, press enter and it will list the shorthand outlines for that word and any related words.

The program not only finds exact matches but also finds words and their corresponding outlines that contain the word you typed.

Special Search Tips

While you can simply enter words as you would in any dictionary search, this program offers additional flexibility through regular expressions. Regular expressions are a way to describe word patterns using special characters. For this program, you can think of them as a helpful tool to find words that match certain patterns. While they might sound complex, you don't need to worry about the technical details. Here's what you need to know:

Find words that start with something:

Type ^ before your word. For example, ^pre will find words like "prefix", "prelude", and "prevent".

Find words that end with something:

Type $ after your word. For example, ing$ will find words like "running", "jumping", and "singing".

Using Characters Inside Brackets:

  • To find words that match any one of several characters, use brackets [].
  • Example: capit[ao]l will match both "capital" and "capitol".

General Tips:

  • Wildcards: Use . to match any single character.
  • Example: p.t will match "pat", "pet", "pit".
  • Repetition: Use * to match zero or more occurrences of the preceding element.
  • Example: pre* will match "pre", "prefix", "preempt".

Examples:

  • Prefix Search: ^un will match "undo", "unite", "universe".
  • Suffix Search: ly$ will match "quickly", "slowly", "happily".
  • Contains Part: cat will match "catalog", "scatter", "concatenate".

Don't worry if these special tricks seem confusing at first. You can always just type in regular words, and the program will still help you find what you're looking for!

If you like using regular expressions but need help, you can always describe the search to AI assistant such as ChatGPt let it create a regular expression for your search. Happy word hunting!
